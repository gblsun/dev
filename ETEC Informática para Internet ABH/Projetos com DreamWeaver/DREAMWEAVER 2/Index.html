InformÁtica para Internet

O curso Informática para Internet disponibilizada pela rede Centro Paula Souza é um curso focado para pessoas com interesse interligado a Internet em geral. Tem aulas técnicas disponibilizadas com tema de Gestão e conteúdo Web, Programação e Algoritmos, Interfaces Web e diversos outros. Além de o aluno poder escolher entre apenas o ensino técnico ou também o ensino técnico junto com o ensino médio.

Gestão de Conteúdo Web 

Gestão de contéudo Web é uma das matérias do ensino técnico mais importantes do curso de Informática para Internet na minha opnião. Isso porque ele ensina desde os principíos básicos de produzir um site utilizando linguagens de progamação e gestando e almejando os conteúdos dentro de um conteúdo web. Isto não é apenas programação.

Segurança da Informação

Segurança da Informação (ou cibersecurity) é uma das aulas que eu estou mais animado para assistir, isso porque eu gosto de segurança da informação. Bom, nessa aula, iremos aprender sobre proteção de um conjunto de valores, no sentindo de preservar o valor que possuem para um indivíduo ou uma organização.

Matemática

Na Etec Adhemar Batista Heméritas (etec que estou estudando) é uma escola que disponibiliza cursos técnicos, porém também é disponibilizado aulas do ensino médio, que também são ótimas. Esse é um exemplo de uma delas. Matemática é uma matéria indispensável para qualquer um. Aprendemos Aritmética, geometria, equações e contas.

Química

Química é outra matéria super importante, principalmente para quem quiser prestar vestibular.  Além de ser uma matéria bem divertida de se aprender.
