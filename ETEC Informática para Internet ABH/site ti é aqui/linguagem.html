Linguagens em que produzimos nossos sites

HTML5

HTML5 é um conjunto de tecnologias que você pode usar para criar sites e aplicativos interativos modernos com menos dependência de plugins de terceiros, como Adobe Flash. Em vez disso, você pode usar HTML5, CSS3 e JavaScript para criar aplicativos e sites que são extremamente responsivos, sempre funcionando e que podem ser acessados em qualquer dispositivo.
A principal diferença entre HTML5 e seus predecessores é o foco em aplicações web e a inclusão de diversas API's para acessar diversos recursos do sistema operacional, tais como: armazenamento local, geolocalização, câmera, e diversos outros. Além disso, a linguagem HTML5 introduziu uma série de tags semânticas para melhorar a acessibilidade e a estrutura do conteúdo.


Ao contrário de anteriores versões do HTML, o HTML5 é pensado para dispositivos móveis e pode ser facilmente adaptado para tablets e celulares. Esse é um dos principais motivos pelos quais as empresas estão usando a tecnologia para criar aplicativos e sites. Além disso, os sites criados com HTML5 são extremamente leves e carregam rapidamente, o que é ideal para dispositivos moveis também.

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O HTML não foi projetado para ter tags que ajudariam a formatar a página. Tags como foram introduzidas na versão 3.2 do HTML e causaram muitos problemas para os desenvolvedores. A relação entre HTML e CSS é bem forte. Como o HTML é uma linguagem de marcação e o CSS é focado no estilo , eles andam juntos. CSS não é tecnicamente uma necessidade, mas provavelmente você não gostaria de olhar para um site que usa apenas HTML, pois isso pareceria completamente abandonado.
