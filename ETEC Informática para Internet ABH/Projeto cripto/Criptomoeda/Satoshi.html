Satoshi

Um Satoshi é uma pequena fração do bitcoin, podemos entender melhor isso pensando: 100 centavos é igual a 1 real, então 1 bitcoin são 100.000.000 Satoshi
o bom do Satoshi é não necessariamente ter 1 bitcoin para fazer o uso dele, sendo possível realizar de uma forma mais eficaz por ser uma moeda com menor preço poder realizar diversos tipos de transações.
