## Descrição da Atividade

**Atividade Prática 1 — Dataset Iris**  
Disciplina: Análise de Dados  

Esta atividade tem como objetivo introduzir conceitos fundamentais de **análise exploratória de dados (EDA)** utilizando o **dataset Iris**, um dos conjuntos de dados clássicos da ciência de dados e estatística.

Ao longo da atividade, serão aplicados métodos de **estatística descritiva**, análise de frequência e **visualização de dados**, permitindo compreender melhor a distribuição e as características das variáveis presentes no dataset.

### Objetivos da Atividade

- Realizar o **carregamento e inspeção inicial do dataset Iris**.
- Calcular **frequências absolutas e relativas** da variável *Species*.
- Calcular **medidas de tendência central**:
  - Média
  - Mediana
- Calcular **medidas de dispersão**:
  - Desvio padrão
  - Variância
- Analisar **estatísticas por espécie**, como a média do comprimento da pétala.
- Produzir **visualizações gráficas** para explorar padrões nos dados, incluindo:
  - Gráfico de barras
  - Gráfico de pizza
  - Histograma
  - Boxplot
- Interpretar os resultados obtidos e responder a um **questionário conceitual** sobre os gráficos e estatísticas.

### Entregáveis

Para a entrega da atividade, devem ser compartilhados os seguintes arquivos:

- **Link do notebook no Google Colab** contendo a análise realizada.
- **Arquivo PDF** contendo as respostas do questionário.
- **Arquivo `.xlsx`** contendo outras análises complementares e o dataset utilizado na atividade.
